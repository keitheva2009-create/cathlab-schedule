# 心導管室排程系統 (Cath Lab Schedule)

單一 HTML 檔案的心導管室手術排程系統，取代原本共用的 Google 試算表。

## 特色

- **零安裝**：單一 `index.html`，用 Chrome / Edge 直接開啟即可使用
- **Dropbox 同步**：排程資料存在一個 JSON 檔（透過 File System Access API 讀寫），放在 Dropbox 共享資料夾即可多台電腦同步（每 4 秒自動偵測更新）
- **版面比照原試算表**：兩間導管室並排（Philip / SIEMENS），每日 8:00–12:00、13:00–17:00 各半小時時段＋加台1–4
- **手術計時**：一鍵蓋「手術開始／結束」時間戳，進行中的台以紅底閃爍標示
- **劃線取消**：取消的排程以刪去線保留原格，可復原；時段可再排他人
- **Excel 匯入**：直接讀取從 Google 試算表下載的 .xlsx（自製解析器，無外部函式庫），自動辨識日期／時段／欄位／合併儲存格，匯入前逐筆預覽
- **下拉選單可維護**：術式、醫師、床號/醫院、細項清單可在設定中編輯，新值自動學習
- **列印**：A4 橫向列印當週排程

## 使用方式

1. 下載 `index.html` 與 `使用說明.txt`，放到 Dropbox 內的資料夾（例如 `心導管排程`）
2. 用 Chrome 或 Edge 開啟 `index.html`，第一次點「建立新資料檔…」並將資料檔存在同一資料夾
3. 其他電腦同步後開啟同一份 `index.html`，點「開啟現有資料檔…」選同一個 JSON

詳細操作見 `使用說明.txt`。

## 注意

- **本儲存庫不含任何排程／病人資料**——資料檔（`心導管排程資料.json`）只存在於使用者的 Dropbox，請勿上傳
- 多人協作採「最後儲存者為準」，Dropbox 版本紀錄可作為還原備援
- 需要 Chromium 系瀏覽器（Chrome / Edge）以支援 File System Access API
