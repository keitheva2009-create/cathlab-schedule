心導管室排程系統 — 使用說明
================================

【系統需求】
- 請使用 Chrome 或 Edge 瀏覽器開啟（需要「檔案直接讀寫」功能）。
- 本資料夾放在 Dropbox 內，資料檔會自動同步到所有安裝 Dropbox 的電腦。

【最快的開啟方式（推薦）：網頁＋雲端資料庫】
1. 點網址開啟程式：https://keitheva2009-create.github.io/cathlab-schedule/
2. 第一次開啟時，把「團隊金鑰」貼進最上方的欄位 → 按「連線」
   （金鑰在本資料夾的「團隊金鑰.txt」，或向排程負責人索取）
3. 完成！之後這台電腦／手機會自動連線，不需再輸入。

雲端資料庫說明：
- 排程資料存放於 Google Firestore「台灣機房」，憑 48 碼金鑰存取，
  金鑰請勿外流給團隊以外的人。
- 「即時」同步：任何人的修改幾秒內出現在所有人畫面上，
  多人同時編輯不同病人不會互相覆蓋。
- 手機瀏覽器也可使用（Chrome / Edge / Safari）。
- 每日自動備份：在「⚙ 設定」→「設定每日備份資料夾…」選擇
  Dropbox 的「心導管排程」資料夾，系統每天會自動存一份
  「雲端備份-日期.json」（可用「匯入資料」還原）。
- 舊的「心導管排程資料.json」仍保留作為歷史備份，不再需要手動選擇。

【第一次使用】
1. 用滑鼠雙擊 index.html，以 Chrome 或 Edge 開啟。
2. 點「建立新資料檔…」，把資料檔存在本資料夾內
   （建議檔名：心導管排程資料.json，直接存在「心導管排程」資料夾）。
3. 之後系統會記住這個資料檔；下次開啟時若瀏覽器詢問，
   點「重新連線資料檔」再允許權限即可。

【其他電腦上使用】
1. 等 Dropbox 同步完成後，同樣雙擊該電腦 Dropbox 裡的 index.html。
2. 第一次請點「開啟現有資料檔…」，選擇資料夾中的「心導管排程資料.json」。
3. 之後就會自動連線同一個資料檔，各電腦看到相同排程。
※ 若不小心點成「建立新資料檔…」又選到既有的排程資料檔，
   系統會直接連線使用該檔案，不會覆蓋內容，請放心。

【帳號登入（雲端模式）】
- 連上雲端後會要求登入。第一次登入：帳號與密碼都是你的「員工編號」
  （例如 G00212 / G00212），系統會自動建立帳號。
- 第一次登入後會提示修改密碼（可先按「下次再說」，之後隨時可在
  「⚙ 設定」→「修改密碼」變更）。
- 每台電腦／手機登入一次後會記住；「⚙ 設定」→「登出帳號」可登出。
- 所有的新增、修改、取消、復原、刪除、匯入都會記錄操作帳號與時間：
  「⚙ 設定」→「📋 異動紀錄」可查看最近 300 筆（含被刪除的排程）。
- 點開任何病人也會顯示「建立：帳號（時間）／最後修改：帳號（時間）」。

【日常操作】
- 點某一天、某個時段的「空白格」→ 新增排程。
- 點已排的病人 → 編輯或刪除。

【取消排程（劃線保留）與永久刪除】
- 點病人 → 「✂ 取消排程（劃線保留）」：病人會以「灰色劃線」留在原格，
  大家都看得到「原本有排、後來被取消」，該時段隨即可再排別人。
- 點劃線的名字 → 可查看細節、按「↩ 復原排程」恢復
  （若原時段已被排走，會提示衝突，請先調整）。
- 「刪除（永久）」則完全移除、不留紀錄；一般情況建議用劃線取消。
- 劃線取消的排程會記錄取消時間（滑鼠停在劃線名字上可見）。
- 「佔用時間」可選 30 分鐘～4 小時，會自動合併多個時段（同一診次內）。
- 加台1～加台4 為加台名額（灰底列），一格一台。
- 週一至週六為預設排程日；週日可在「⚙ 設定」開啟。
- 上方可切換上週／本週／下週，或用日期選擇器跳到任何一週。
- 每 4 秒會自動檢查資料檔，其他電腦的修改會自動出現。

【手術開始／結束（自動時間戳）】
- 點排程中的病人 → 按「▶ 現在開始」：自動蓋上現在時間並儲存，
  該台在表格上會變成紅底並顯示閃爍的「● 手術中 HH:MM~」標示。
- 手術完成後再點該病人 → 按「■ 現在結束」：蓋上結束時間，
  表格改顯示綠色「✔ HH:MM~HH:MM」（滑鼠停在姓名上也會顯示完整時間）。
- 按錯時間可用時間旁的「✕」清除後重蓋。
- 時間戳會寫入資料檔，所有電腦同步看得到目前哪一台正在進行。

【從 Google 試算表匯入舊排程】
1. 在 Google 試算表：檔案 → 下載 → Microsoft Excel (.xlsx)。
2. 在本系統點右上角「📥 匯入Excel」→ 選擇剛下載的檔案。
3. 系統會自動辨識日期（星期X 月/日）、時段、左右兩室與各欄位，
   跨列合併的儲存格會自動換算成佔用時段；先顯示預覽再匯入。
4. 確認「年份」選對（試算表上通常只有月/日；若星期對不上會提醒；
   跨年的週（12月接1月）會自動換年，不必分兩次匯入）。
5. 有多個工作表（分頁）時，可在「工作表」下拉逐一選擇、分次匯入。
※ 預覽表會先標示每一筆的狀態：「將匯入」／「重複，略過」／「衝突，略過」，
   確認無誤再按匯入；讀不懂的列（缺時段、缺日期等）都會在黃色區塊列出原因。
※ 性別欄的「男／女」會自動轉成 M／F；新的術式、醫師名稱會自動加入下拉選單。

【分享給同事】
- 點右上角「🔗 分享」會複製一段訊息（含程式位置說明），
  直接貼到 LINE／Email 給同事即可。
- 對方電腦需已同步同一個 Dropbox 資料夾，開啟其中的 index.html。

【下拉選單維護】
- 點右上角「⚙ 設定」，可編輯：術式、醫師、床號/醫院、細項 的選單內容
  （每行一項），也可改導管室名稱、顯示週六/週日。
- 排程時輸入了選單中沒有的新項目，系統會自動加入選單。

【備份與匯入】
- 「⚙ 設定」→「匯出備份」：下載一份 JSON 備份檔。
- 「⚙ 設定」→「匯入資料…」：從備份檔還原（會覆蓋目前全部資料）。
- Dropbox 本身也保留檔案版本紀錄，可在 Dropbox 網頁版還原舊版資料檔。

【注意事項（多人同時使用）】
- 系統採「最後儲存者為準」。請盡量避免兩個人「同一時間」修改排程；
  若 Dropbox 偵測到衝突，會產生「衝突的複本」檔案，
  屆時請人工比對後保留正確的那份。
- 資料檔（.json）請勿用 Word/Excel 開啟編輯。

【列印】
- 點右上角「🖨 列印」，會以 A4 橫向列印目前顯示的一週。
